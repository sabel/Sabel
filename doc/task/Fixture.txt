フィクスチャ
------------------------------------------------------------------------------

既存のデータベース内容からFixtureクラスを作成する

    % sakle Fixture development import モデル名
    % sakle Fixture development import all # 全テーブルで実行
    
データベースへFixtureを適応する
    
    % sakle Fixture development run モデル名
    
コマンド名 sakle
タスク指定 Fixture
環境指定 (development|test|production)
実行コマンド (run|import)
オプション
  run
    モデル名
  import
    モデル名
    all
