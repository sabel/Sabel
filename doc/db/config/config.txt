sabelコマンドでアプリケーションの雛型を作成すると、configディレクトリの下に'connnection.php'というファイルがあります。初期パラメータは以下のように設定されています。

  $params = array("default" => array(
                    "driver"   => "mysql",
                    "host"     => "localhost",
                    "database" => "default",
                    "user"     => "root",
                    "password" => "")
                 );

この$paramsのキーがコネクションネームとなります。
'default'というコネクションネームはいくつかの場面で省略することができます。

設定できるパラメータには以下ものがあります。

[driver]
使用するドライバ名を指定します。
sabel.dbが標準で実装しているドライバは以下のものになります。

  * mysql - PHP MySQL
  * mysqli - PHP MySQLi
  * pdo-mysql - PDO_MYSQL
  * pgsql - PHP PostgreSQL
  * pdo-pgsql - PDO_PGSQL
  * pdo-sqlite - PDO_SQLite
  * ibase - PHP Firebird/InterBase
  * oci - PHP OCI
  * mssql - PHP Microsoft SQL Server (experimental)

* PDOを使用する場合はPDOエクステンションが必要です。

[host]
接続するデータベースのホスト名、またはIPアドレスを指定します。

[database]
使用するデータベース名を指定します。

[user]
データベースへ接続するユーザ名を指定します。
* OracleではOWNERと同じである必要があります。

[password]
上で指定したユーザのパスワードを指定します。

[port]
データベースのポート番号を指定します。
データベース側で明示的にポートを変更した場合はこれを設定する必要があるでしょう。

[schema]
使用するスキーマ名を指定します。
* MySQLでは上記の[database]が使用されます。
* PostgreSQLでは指定しない場合'public'が使用されます。
* Oracleでは上記の[user]が使用されます。

[encoding]
使用するエンコーディング名を指定します。
